# Design System dla swap.lol

## Filozofia Designu

### Kolorystyka
- **Primary**: Głęboki czarny (#0a0a0a) jako tło główne
- **Secondary**: Ciemny grafit (#1a1a1a) dla kart i sekcji
- **Accent**: Neonowy cyjan (#00ffff) dla akcentów i CTA
- **Accent 2**: Neonowy fiolet (#8a2be2) dla dodatkowych elementów
- **Text Primary**: Biały (#ffffff) dla głównego tekstu
- **Text Secondary**: Jasnoszary (#a0a0a0) dla tekstu pomocniczego

### Typografia
- **Display Font**: "Orbitron" - futurystyczna czcionka dla nagłówków
- **Body Font**: "Inter" - nowoczesna, czytelna czcionka dla treści
- **Monospace**: "JetBrains Mono" dla kodu i specjalnych elementów

### Wzory i Tekstury
- **Geometric patterns**: Subtelne siatki i linie w tle
- **Glow effects**: Neonowe poświaty dla akcentów
- **Gradient overlays**: Płynne przejścia kolorów
- **Glass morphism**: Przezroczyste elementy z rozmyciem

## Style CSS
- **Border radius**: 12px dla kart, 8px dla przycisków
- **Shadows**: Deep black shadows z neonowymi glowami
- **Transitions**: Smooth 0.3s ease dla wszystkich interakcji
- **Backdrop blur**: 10px dla efektu szkła

### Przyciski i Interakcje
- **Primary Button**: Neonowy cyjan z czarnym tekstem
- **Secondary Button**: Przezroczysty z neonową ramką
- **Hover effects**: Glow i lekkie powiększenie
- **Active states**: Głębszy kolor i inner shadow

## Layout
- **Grid system**: CSS Grid i Flexbox
- **Container max-width**: 1200px
- **Responsive breakpoints**: Mobile-first
- **Spacing**: 8px base unit system
