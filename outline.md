# Struktura strony swap.lol

## Strony:

### 1. index.html - Strona Główna
- **Hero Section**: 
  - Animowany nagłówek z efektem typewritera
  - Podtytuł z opisem usługi
  - CTA przyciski (Sign Up, View Pricing)
  - Tło z animowanymi particle'ami

- **Features Section**:
  - Karty z funkcjonalnościami
  - Ikony i opisy
  - Animacje przy scrollu

- **Stats Section**:
  - Liczniki statystyk
  - Social proof
  - Live counter

- **Example Profiles**:
  - Karty przykładowych profili
  - Mini-preview funkcjonalności

### 2. pricing.html - Cennik
- **Pricing Cards**:
  - 3 plany (Free, Premium, Pro)
  - Porównanie funkcji
  - Toggle monthly/yearly
  - Animacje i hover effects

- **FAQ Section**:
  - Rozwijane pytania
  - Kategorie FAQ

- **CTA Section**:
  - Zachęta do rejestracji
  - Gwarancje i zaufanie

### 3. profiles.html - Customizacja Profili
- **Template Gallery**:
  - Grid szablonów
  - Preview na żywo
  - Kategorie stylów

- **Customization Panel**:
  - Color picker
  - Font selector
  - Layout options
  - Live preview

- **Profile Builder**:
  - Step-by-step wizard
  - Drag & drop
  - Real-time preview

### 4. dashboard.html - Panel Użytkownika
- **Stats Dashboard**:
  - Analytics
  - Views counter
  - Link tracking

- **Profile Management**:
  - Lista profili
  - Quick edit
  - Duplicate/Delete

- **Settings**:
  - Account settings
  - Privacy options
  - Billing

## Komponenty:
- Navigation bar (sticky)
- Footer z linkami
- Modal system
- Toast notifications
- Loading states
- Error boundaries
